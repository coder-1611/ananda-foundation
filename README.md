# Ananda Foundation — Website

Static website for Ananda Foundation (Sikariya, Nawada, Bihar), exported from Claude Design.

## Structure
- `ananda/` — the multi-page site (`index.html` is the homepage) + `style.css`
- `Ananda Foundation.html` — self-contained single-page version of the same site
- `uploads/` — image assets referenced by the pages

## Note on images
The pages reference 8 photos in `uploads/` (hero, logo, program/product/gallery images).
These are added from the original project export.
